以下のように指定すると言語ごとの引用符を設定できる。 
q:lang(ja) {
  quotes: "「" "」" "『" "』";
}

これは他のタグにも設定でき、spanタグに指定したい場合は次のようにする。 
span.quote:lang(ja) {
  quotes: "「" "」" "『" "』";
}
span.quote:before {
  content: open-quote;
}
span.quote:after {
  content: close-quote;
}
